# Registry MCP

The registry MCP is future design/spec-only material for a possible read-only server that could discover toolkit metadata.

This is MCP-ready registry and design/spec material. It is not a runnable MCP server.

It consumes JSON registries with `JSON.parse` and exposes discovery/routing only.

No shell execution. No arbitrary file writes. No arbitrary file reads outside approved toolkit content.
